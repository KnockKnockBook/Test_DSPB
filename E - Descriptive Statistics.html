Descriptive Statistics

~~here is the content~~

:::python print("The triple-colon syntax will not show line numbers.")

Create an example DF
import pandas as pd
import numpy as np
DF = pd.DataFrame()

DF['A'] = [1, 2 ,3]
DF['B'] = ['a','b', 'b']
DF['C'] = ['how',  'about','that']
Descriptive Statistics: Numerics
DF['A'].describe()

DF['A'].sum()         # the sum total of all values in the column
DF['A'].count()       # Count non null values
DF['A'].mean()        # the average
DF['A'].median()      # Median - the middle value 
DF['A'].mode()        # Mode - most frequently occuring value
DF['A'].std()         # Standard deviation
DF['A'].min()         # Min
DF['A'].quantile(.25) # Quantile 25% 
DF['A'].quantile(.50) # Quantile 50%  
DF['A'].quantile(.75) # Quantile 75% 
DF['A'].max()         # Max
Descriptive Statistics: Categorical columns
# count number of distinct values in column 'B'
DF.groupby('B')['A'].nunique()
